# 🌿 Bazar de la Magnolia — Huerto Creativo

Aplicación web full-stack para la gestión y venta de productos naturales, arte y conservas, inspirada en tiendas reales tipo vivero boutique.

Incluye:
- Catálogo de productos (plantas, arte, conservas)
- Sistema de carrito de compras
- Gestión de stock
- Panel admin básico
- Backend con MySQL
- Frontend en React

---

## 🚀 Tecnologías utilizadas

### Frontend
- React
- React Router
- CSS personalizado (sin frameworks)
- Fetch API

### Backend
- Node.js
- Express
- MySQL (mysql2/promise)
- dotenv
- crypto (UUID)

### Base de Datos
- MySQL (modelo relacional padre/hijo)
